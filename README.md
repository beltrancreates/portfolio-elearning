# Cartógrafo del Significado

Un portfolio multipágina que representa la identidad de un diseñador de aprendizaje como un navegante en busca de sentido en un mar de complejidad.

## Estructura del Proyecto

```
cartografo-del-significado/
├── index.html                 # Portada-manifiesto
├── cartografo.html            # Manifiesto del cartógrafo
├── proyectos.html             # Proyectos como islas
├── bitacora.html              # Bitácora de ideas
├── travesia.html              # Mapa interactivo
├── contacto.html              # Contacto
├── css/
│   ├── style.css              # Estilos globales
│   ├── home.css               # Estilos de portada
│   └── pages.css              # Estilos de páginas internas
├── js/
│   ├── main.js                # Scripts globales
│   └── travesia.js            # Interactividad del mapa
├── assets/
│   ├── img/
│   │   └── README.txt
│   └── svg/
│       └── README.txt
└── README.md
```

## Cómo Ejecutarlo Localmente

1. Clona o descarga el proyecto.
2. Abre `index.html` en un navegador web moderno.
3. Navega entre páginas usando la barra de navegación.

No requiere servidor local ni dependencias externas.

## Representación de la Web

Esta web modela la identidad profesional como una travesía:
- **Mar**: Complejidad del conocimiento
- **Rutas**: Procesos de comprensión
- **Faros**: Claridad pedagógica
- **Islas**: Territorios de proyectos
- **Barquero**: El autor como guía

## Donde Reemplazar Imágenes o Textos

- `assets/img/`: Añade texturas como `texture-paper.png`
- `assets/svg/`: Iconos SVG personalizados
- Contenido textual: Edita directamente en los HTML
- Tipografía: Georgia (serif) para elementos artísticos, system-ui para texto general
